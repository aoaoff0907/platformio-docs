.. _cmd_system_prune:

platformio system prune
=======================

.. versionadded:: 5.0

.. contents::

Usage
-----

.. code-block:: bash

    platformio system prune
    pio system prune

Description
-----------

Remove unused data:

- cached API requests
- cached package downloads
- temporary data.

Options
-------

.. program:: platformio system prune

.. option::
    --force, -f

Do not prompt for confirmation.

Examples
--------

.. code::

    > platformio system prune

    WARNING! This will remove:
     - cached API requests
     - cached package downloads
     - temporary data
    Do you want to continue? [y/N]: y
    Total reclaimed space: 36.48KB
